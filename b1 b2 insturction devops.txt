B1: Create Docker Image and Run Application
🎯 Aim

To create a Docker image from a local Python application and run it using a Docker container.

🧰 Tools Required
Docker Desktop
Python (optional, for writing app)
Command Prompt / Terminal
📁 Project Structure
myapp/
│── app.py
│── Dockerfile
🧩 Step 1: Install Docker
Download Docker Desktop
Install and run as Administrator
Verify installation:
docker --version
⚠️ Problem Faced
DockerDesktop must be owned by an elevated account
✔ Solution
Delete folder:
C:\ProgramData\DockerDesktop
Reinstall Docker as Administrator
🧩 Step 2: Create Python Application
📄 app.py
print("Hello from Docker 🚀")
🧩 Step 3: Create Dockerfile
📄 Dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY . .
CMD ["python", "app.py"]
🧩 Step 4: Build Docker Image

Open terminal in project folder:

docker build -t myapp .
⚠️ Problem Faced
docker not recognized
✔ Solution
Restart system
Ensure Docker Desktop is running
🧩 Step 5: Verify Image
docker images

Expected:

myapp   latest
🧩 Step 6: Run Docker Container
docker run myapp
✅ Output
Hello from Docker 🚀


☁️ STEP 4: Push Image to Docker Hub
🔹 Login
docker login -u surajkamathh

👉 Use Access Token (NOT password)

🔹 Tag image
docker tag myapp surajkamathh/myapp:v1
🔹 Push image
docker push surajkamathh/myapp:v1
⚠️ Problems faced
Issue	Fix
Wrong push command	Use correct format
Password not working	Use Access Token
Token error	Regenerate token
🌐 STEP 5: Push Code to GitHub
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/yourusername/repo.git
git push -u origin main
⚙️ STEP 6: Add Jenkinsfile
📄 Create Jenkinsfile
pipeline {
    agent any

    environment {
        IMAGE_NAME = "surajkamathh/myapp"
        IMAGE_TAG = "v1"
    }

    stages {

        stage('Docker Check') {
            steps {
                bat "docker --version"
            }
        }

        stage('Login to Docker Hub') {
            steps {
                withCredentials([usernamePassword(
                    credentialsId: 'dockerhub-creds',
                    usernameVariable: 'DOCKER_USER',
                    passwordVariable: 'DOCKER_TOKEN'
                )]) {
                    bat '''
                    docker logout
                    docker login -u %DOCKER_USER% -p %DOCKER_TOKEN%
                    '''
                }
            }
        }

        stage('Build Docker Image') {
            steps {
                bat "docker build -t %IMAGE_NAME%:%IMAGE_TAG% ."
            }
        }

        stage('Push Docker Image') {
            steps {
                bat "docker push %IMAGE_NAME%:%IMAGE_TAG%"
            }
        }
    }
}
🔹 Push Jenkinsfile
git add Jenkinsfile
git commit -m "added pipeline"
git push
⚙️ STEP 7: Setup Jenkins
🔹 Create Pipeline Job
Jenkins → New Item → Pipeline
🔹 Configure
Pipeline script from SCM
Git repo URL
Script path:
Jenkinsfile
🔐 STEP 8: Add Docker Credentials in Jenkins

Go to:

Manage Jenkins → Credentials

Add:

Kind: Username with password
Username: surajkamathh
Password: Docker Access Token
ID:
dockerhub-creds
⚠️ Problems faced
Issue	Fix
Credentials not found	ID mismatch
Login failed	Wrong token
Token stdin issue	Use -p
